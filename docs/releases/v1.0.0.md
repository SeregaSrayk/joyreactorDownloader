First public release of **joyreactorDownloader** — кросс-платформенное (Windows / macOS / Linux) десктоп-приложение для пакетного скачивания картинок с [joyreactor.cc](https://joyreactor.cc) по фильтрам.

Стек: Go + [Wails v2](https://wails.io). Один статический бинарник, без внешних рантаймов. Использует **официальный GraphQL API** Joyreactor — никакого HTML-скрейпа.

## Что внутри

### Поиск и фильтры
- Свободный текст, теги (с исключениями), автор, диапазон рейтинга
- Флаги NSFW / unsafe / «только избранное» (последнее требует логин)
- Клиент-сайд фильтры: тип медиа (image / gif / video), мин. ширина-высота, диапазон дат, лимит файлов
- Автодополнение тегов с индикатором NSFW и числом постов
- Валидация автора через `Query.user(name:)`
- Авто-учёт **заблокированных тегов** из профиля для залогиненных

### Авторизация и состояние
- Cookie-based вход через GraphQL `login`-мутацию. Пароль **никогда** не сохраняется.
- Сессия восстанавливается между запусками (`%APPDATA%/joyreactorDownloader/session.json`)
- **Пресеты фильтров с привязкой папки** — последний пресет автовосстанавливается при следующем запуске
- Настройки приложения и размер окна сохраняются в config-папке ОС

### Скачивание
- Параллельные воркеры внутри задачи + несколько задач одновременно (GraphQL-клиент сериализует API-запросы автоматически)
- Пауза / возобновление / отмена на уровне отдельной задачи
- Атомарная запись на диск (`.part` → rename)
- **Два режима манифеста дедупликации** (в Settings): per-folder (по умолчанию) или общий для всех папок
- Два формата имён файлов:
  - `<postId>_<attrId>.<ext>`
  - `[tag1][tag2][tag3]_<postId>_<attrId>.<ext>` — теги сортируются алфавитно, так что один и тот же пост всегда даёт одинаковое имя

### Превью UI
- Сетка превью с бейджами: NSFW, тип медиа, число картинок в посте, «уже скачано» (зелёная галочка)
- ЛКМ по плитке открывает full-screen превью поста с картинками и комментариями
- В тексте комментариев работают ссылки (`<a>`), переводы строк и базовое форматирование (`<b>/<i>/<em>/<strong>`); остальной HTML вырезается
- Теги-пилюли в превью кликабельны: подсвечивает уже добавленные, клик добавляет/убирает в фильтр
- ПКМ по картинке вызывает нативное контекстное меню WebView («Сохранить изображение как…», «Копировать изображение» и т.п.)
- При «Сохранить как…» предлагает имя файла **в выбранном тобой формате**
- Анимированные GIF проигрываются как webm-транскод (когда у JR он есть); если 404 — graceful fallback на исходный `.gif`

### Ручной выбор
- Тоггл «Ручное скачивание» в шапке превью включает чекбоксы на плитках
- Резинка-выделение мышкой (drag-select) для batch-выбора
- Уже скачанные посты нельзя выбрать — чекбокс скрывается
- Кнопка «＋ Сохранить» прямо в overlay-превью для одиночного поста
- Выбор сбрасывается при новом поиске / смене пресета

### Очередь
- Компактная модалка по кнопке `📋 Очередь N` в шапке
- Статус, имя, папка, прогресс-бар, счётчики (скачано `✓` / пропущено `⊘` / ошибок `✖`), действия для каждой задачи
- Системные тосты при завершении (Windows нативные, на macOS/Linux отключены)

### Прочее
- Single-instance lock (повторный запуск exe фокусирует уже открытое окно)
- Кросс-платформенная «Открыть папку» (`explorer` / `open` / `xdg-open`)
- Запоминается размер окна и режим «развернуть на всё доступное пространство»

## Установка

Скачай zip для своей платформы ниже:

| Платформа | Файл | Внутри |
|---|---|---|
| Windows | `joyreactorDownloader-v1.0.0-windows.zip` | `joyreactorDownloader.exe` |
| macOS | `joyreactorDownloader-v1.0.0-macos.zip` | `joyreactorDownloader.app` |
| Linux | `joyreactorDownloader-v1.0.0-linux.zip` | ELF-бинарник `joyreactorDownloader` |

**Системные требования:**
- Windows 10/11: WebView2 (предустановлен в Win11, в Win10 ставится одной кнопкой при первом запуске)
- macOS 11+: WKWebView (предустановлен)
- Linux: `libwebkit2gtk-4.0-37` + `libgtk-3-0` (стоят по умолчанию в Ubuntu 22.04, Fedora 38, Debian 12)

## Сборка из исходников

```sh
go install github.com/wailsapp/wails/v2/cmd/wails@v2.12.0
cd cmd/joyreactor-gui
wails build -clean
# → cmd/joyreactor-gui/build/bin/joyreactorDownloader[.exe|.app]
```

Все частые команды собраны в `Makefile` (`make build`, `make dev`, `make test`, `make check`).

## Что НЕ умеет

- Не обходит капчу / антибот-проверки
- Не делает мульти-аккаунтное скачивание
- Не парсит HTML — только GraphQL API
- Нет встроенного proxy/VPN — использует прямое сетевое соединение

Для личного использования с разумными лимитами.
